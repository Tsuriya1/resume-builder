# Resume Builder

Resume Builder is a web application for creating, editing, tailoring, styling, and exporting professional resumes.

It is designed primarily for individual job seekers, with strong emphasis on software engineers, developers, and technical professionals.

The system supports:

- structured resume editing
- multiple resume versions
- live preview
- template-based styling
- AI-assisted rewriting
- job tailoring
- export to PDF and DOCX
- English and Hebrew support
- Google authentication
- admin management
